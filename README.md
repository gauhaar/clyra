# CLYRA — Operational Control for Modern Construction Companies

Clyra is a structured operational management system designed specifically for construction companies that require clarity, accountability, and financial control across complex projects.

This repository contains the official marketing landing page built to communicate Clyra’s strategic value to executives, investors, and construction leadership teams.

---

## 🧠 Product Vision

Construction companies don’t fail because of lack of tools.  
They fail because of fragmented visibility.

Clyra connects:

- Company hierarchy  
- Project lifecycle  
- Task execution  
- Financial oversight  
- Performance metrics  

Into one structured operational system.

The landing page reflects this philosophy:
**Clarity → Structure → Control → Measurable Performance**

---

## 🎯 Purpose of This Landing Page

This is not a generic SaaS website.

The landing page is designed to:

- Communicate real operational pain points in construction
- Position Clyra as a structural alternative to tools like Jira and Teams
- Demonstrate financial and performance transparency
- Present executive-level control as the core value proposition
- Convert decision-makers through strategic storytelling

Each section follows a deliberate narrative architecture:

1. Hero — Strategic positioning  
2. Problem — Operational fragmentation  
3. Solution — Structured system  
4. How It Works — Architectural workflow  
5. AI Automation — Intelligent task activation  
6. Financial Control — Budget governance  
7. Performance Intelligence — Measurable execution  
8. CTA — Leadership decision moment  

---

## 🏗 Tech Stack

The landing page is built using:

- **Next.js (App Router)**
- **JavaScript (no TypeScript)**
- **Tailwind CSS**
- **shadcn/ui**
- **Framer Motion**
- Optional 3D visualization via React Three Fiber (where applied)

Architecture is component-based and optimized for clarity and scalability.

This is the Clyra landing page built with Next.js (App Router), Tailwind CSS, shadcn/ui, and Framer Motion.

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open http://localhost:3000 in your browser to see the result.

Edit `app/page.js` to change the landing page content. The page auto-updates as you edit the file.

To build the project for production:

```bash
npm run build
```
